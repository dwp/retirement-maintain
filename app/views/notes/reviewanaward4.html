Scenario

  • Agent reviews an award where the award amount goes down, and the period of the award change covers an annual uprating of the award. The agent is then presented with the option to refer the overpayment to the debt management service.

Business requirements

  • The agent needs to see the new award amounts and the dates those awards are applicable from
  • The agent needs to be able to change the from dates if required
  • The agent can choose to not do the overpayment referral as part of this task, and a separate task is created
  • The agent is presented with all of the relevant information they need to submit a debt referral via the e-referral tool
